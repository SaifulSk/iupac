📐 IUPAC Rules for Naming Organic Compounds

Rule 1 — Find the Longest Carbon Chain

Identify the longest continuous chain of carbon atoms. This gives the parent name: meth(1) eth(2) prop(3) but(4) pent(5) hex(6) hept(7) oct(8) non(9) dec(10)

Rule 2 — Identify the Principal Functional Group

Priority order (highest to lowest): -COOH > -CHO > -CO- > -OH > C=C > C≡C > -X. The chain must include this group.

Rule 3 — Number the Chain

Number from the end nearest to the principal functional group. For alkanes with only substituents, number from end nearest the first substituent.

Rule 4 — Name Substituents as Prefixes

Substituents listed alphabetically with locant numbers: 1-bromo, 2-chloro, 3-methyl.... Identical groups use multipliers: di, tri, tetra

Rule 5 — Suffix for Functional Group

Alkane → -ane | Alkene → -ene | Alkyne → -yne | Alcohol → -ol | Aldehyde → -al | Ketone → -one | Carboxylic acid → -oic acid
